Automated Low-Tech Greenhouse System

Design and development of an intelligent ventilation system for greenhouses. The project aims to optimize plant growth conditions through automated climate control, using a "low-tech" approach focused on efficiency, durability, and cost-effectiveness.

Arduino / C++ Core Technology
Embedded Systems System Architecture
Mechatronics Project Domain

Project Context

In modern agriculture, maintaining an optimal climate is crucial for crop yields. As the Project Leader, I coordinated the development of a robust solution that automates temperature and humidity regulation.

We focused on a "Low-Tech" philosophy: utilizing stepper motors and a custom-built mechanical folding window system to reduce energy consumption while maintaining high reliability.

Technical Contributions

  • C++ Programming: Development of a modular firmware using a non-blocking architecture.
  • Automate's Architecture: Implementation of a Finite State Machine (FSM) to handle safety transitions.
  • Electronic Design: Integration of DHT22 sensors, LCD interfaces, and A4988 motor drivers.
  • Hardware Debugging: Signal analysis to ensure stable communication between sensors and the MCU.

Deep Dive: Software Logic

The core of the system relies on a C++ State Machine. This ensures that the greenhouse doesn't oscillate between states (opening/closing) when the temperature is at the threshold. I implemented a hysteresis logic and timed delays to preserve the mechanical integrity of the motors.
