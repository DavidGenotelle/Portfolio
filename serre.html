Automated Low-Tech Greenhouse System

Design and development of an intelligent ventilation system as part of a mechatronics project. The system utilizes automated folding windows to regulate temperature and humidity, enhancing plant growth conditions through precise environmental control.

C++ / Arduino Tech Stack
FSM Logic Architecture
Automated Ventilation

Project Context

Maintaining an optimal climate is vital for greenhouse productivity. This project focuses on a "Low-Tech" yet highly efficient approach to automate climate regulation.

As the Project Leader, I was responsible for the global coordination of the team and the technical integration between the mechanical components and the embedded system. I worked extensively on the electronics design, ensuring robust signal processing from the sensors.

Project Objectives

  • Design a reliable automated ventilation mechanism
  • Implement real-time monitoring of temperature and humidity
  • Develop a modular C++ codebase for easy maintenance
  • Ensure system durability using a Finite State Machine
  • Optimize stepper motor control for window folding

Technical Approach & Automate's Architecture

The system's intelligence is built on a custom C++ Automate (Finite State Machine). This architecture allows for stable transitions between states (Idle, Opening, Closing, Alert), incorporating hysteresis logic to avoid mechanical wear caused by rapid oscillations.

The electronic design features high-precision DHT sensors and A4988 drivers to control the stepper motors. Every aspect of the firmware was optimized for reliability, featuring manual override capabilities and an LCD interface for local monitoring.
